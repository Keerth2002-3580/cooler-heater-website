# Hero Image Setup

## Required Image

The hero section expects an image at:
```
/public/images/hero-ac.jpg
```

## Image Requirements

- **Filename**: `hero-ac.jpg`
- **Recommended dimensions**: 1920x1080px or higher
- **Aspect ratio**: 16:9 (landscape)
- **Subject**: Modern home interior with air conditioning unit installed
- **Quality**: High resolution (for retina displays)
- **File size**: Optimized for web (under 500KB recommended)

## Where to Find Images

You can use:
1. Your own professional photography
2. Stock photo sites like:
   - Unsplash (https://unsplash.com/s/photos/air-conditioning-interior)
   - Pexels (https://www.pexels.com/search/air%20conditioner/)
   - Shutterstock (premium)

## Search Terms

- "modern home air conditioning"
- "luxury interior with AC unit"
- "contemporary living room air conditioner"
- "premium home cooling system"
- "wall mounted AC unit interior"

## Installation

1. Download your chosen image
2. Rename it to `hero-ac.jpg`
3. Place it in this directory (`/public/images/`)
4. The Next.js Image component will automatically optimize it

## Temporary Solution

If you need to test without an image, you can:
1. Use a placeholder from Unsplash directly (update HeroSection.tsx)
2. Or use a solid color background temporarily

The component is already configured and ready to use the image once you add it!
